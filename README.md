# Welcome to TAFE Mobile Application Assignment

## NSW Traffic Incidents

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Install packages

```bash
   npm install express cors dotenv
   npm install -D typescript ts-node nodemon @types/node @types/express @types/cors
```

3. Start the app

   ```bash
   npx expo start
   ```

## app/(tabs)/home/index

Welcome page and links to go to the pages for Search NSW Traffic Incidents, Region, Type of Incidents, Date and Street Name, View all Incidents Display Page and Saved Incidents Display Page.

## app/(tabs)/home/search-incidents

Search NSW Traffic through Region, Type of Incidents, Street Name and Date.

## app/(tabs)/home/incidents

Displays all NSW Traffic Incidents

## app/(tabs)/home/saved-incidents

Displays all saved NSW Traffic Incidents
